Core Features

Ephemeral Data

Your mailing list is automatically deleted once its time is up. No lingering data, ever.

Anonymous Communication

Communicate without revealing your identity. Privacy is our priority.

Secure & Encrypted

All data is encrypted using modern, industry-standard techniques.

Why FMRL Ink?

  • Privacy First: No persistent storage—your data vanishes when you’re done.
  • Simplicity: Create and manage mailing lists with a single email.
  • Trustworthy: Built on secure infrastructure with end-to-end encryption.

How It Works

  1. Create: Send an email with a subject like "3 days" to anykeyword@fmrl.ink to create your mailing list.
  2. Notify: CC friends to join the list. No registration, no fuss.
  3. Communicate: All messages are relayed anonymously to active subscribers.
  4. Expire: Once the time is up, the list is automatically deleted.
